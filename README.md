## Description

Simple API with NestJS and MongoDB.

## Installation

```bash
$ yarn install
$ docker-compose up --build -d db
$ yarn start:dev
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
